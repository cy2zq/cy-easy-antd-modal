## 快速上手

### 安装

推荐使用 `pnpm` 安装

```bash
pnpm i cy-easy-antd-modal -S
```

### 使用

```tsx | pure
import { Button } from 'antd';
import Modal from 'cy-easy-antd-modal';

export default () => (
  <Modal title="cy-easy-antd-modal" trigger={<Button type="primary">Click Me</Button>}>
    I ❤️ antd
  </Modal>
);
```
